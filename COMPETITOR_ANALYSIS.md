# Competitor Analysis: SundayGrids vs PowerNetPro

## SundayGrids Key Features

### 1. Interactive Savings Calculator
- Slider-based input (avg power bill)
- Real-time calculations
- Shows: Monthly savings, Reserved Solar, Energy/Month, Annual Savings, Long-term savings
- One-time reservation fee display
- "Get Started for Free" CTA

### 2. Comparison Table
- Digital Solar vs Rooftop Solar side-by-side
- Clear visual distinction
- Key differentiators highlighted

### 3. Real-time Stats
- Digital Solar Installed (kW)
- Clean Energy Delivered (kWh)
- Credits Generated (₹)
- Carbon Avoided (kg)

### 4. Testimonials
- Customer quotes with names
- Publication logos
- Carousel/slider format

### 5. Step-by-Step Process
- 3 clear steps with visuals
- Numbered steps
- Images for each step

### 6. Trust Indicators
- Partner logos (Zerodha, IIMB, Tata Power Solar, Social Alpha)
- Clean presentation

### 7. Features Section
- "Zero hassles"
- "No added costs"
- "Diversify your solar"

### 8. FAQ Section
- Categorized (General, Reserving Solar, Using Credits)
- Expandable accordion
